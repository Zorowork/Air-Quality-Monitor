# Air-Quality-Monitor

## Hardware
- Arduino UNO R4 WiFi
- Breadboard
- MQ-2 Gas sensor
- DHT11 Temperature Sensor
- Wires

## Setup
1. Connect the 5V and Ground to Breadboard
2. Connect the MQ-2 and DHT11 to the breadbored
3. Link the MQ-2 and DHT11 to the positive and negative on the breadbored
4. Connect the AO on the MQ2 to the A0 on the Arduino analogue input
5. Connect the DHT11 output to the digital pin 2 on the Arduino
6. Download the code and change the internets name and password
7. Connect the Arduino to computer and upload the code
8. Bluetooth should appear using a bluetooth app (I used nRF Connect)
